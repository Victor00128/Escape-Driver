# Escape Driver

juego de persecucion policial con sistema de estrellas tipo gta

## como jugar

- flechas o wasd para mover
- espacio para drift
- P para pausar
- recoge monedas para puntos
- los powerups te dan ventajas
- haz que los policias choquen entre si

## powerups

- turbo: velocidad x1.8
- escudo: no te hacen daño
- iman: atrae monedas
- bomba: destruye un policia

## dificultades

- normal: 4 policias, 2 min
- dificil: 6 policias, 3 min  
- imposible: 8 policias, 4 min

## instalacion

```
npm install --legacy-peer-deps
npm run dev
```

abre localhost:3000

## tecnologias

- react
- typescript
- vite
- canvas
- web audio api

## tips

- usa drift para girar rapido
- el multiplicador sube si haces cosas seguidas
- los policias pueden chocar entre ellos si los haces ir uno contra otro
